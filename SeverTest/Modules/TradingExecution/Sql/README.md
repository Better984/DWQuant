该目录用于存放本模块的 SQL 脚本，请按功能接口或功能点拆分文件。

## 当前脚本
- `trade_recovery_task.sql`
  - 交易恢复任务持久化表。
  - 用于持久化“平仓写库恢复 / 开仓补偿恢复”任务，支持服务重启后继续恢复执行。
  - 脚本为幂等写法（`CREATE TABLE IF NOT EXISTS`），可重复执行。
- `20260223_order_open_attempt.md`
  - 开仓尝试记录表（成功/失败/上限阻断）。
  - 记录信号时间/信号价/持仓上限快照，支持追溯“命中但被阻断”的开仓。
  - 连续失败统计仅基于真实下单事件，不把上限阻断计入失败暂停。
  - 启动时自动创建表并幂等补列。
