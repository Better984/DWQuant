# AdminBroadcast 模块功能说明

## 功能概述
- 管理员系统广播（系统级通知）。
- 统一管理员权限校验与广播参数校验。
- 服务器配置与运行时间模板管理。
- 超级管理员测试工具（创建测试用户、给指定用户创建随机策略并切换 `testing`）。

## 主要入口
- 控制器：`Controllers/AdminBroadcastController.cs`、`Controllers/AdminServerController.cs`、`Controllers/AdminPositionRiskController.cs`
- 配置与模板：`Controllers/AdminServerConfigController.cs`、`Controllers/AdminRuntimeTemplateController.cs`
- 用户管理与超级测试台：`Controllers/AdminUserController.cs`
- 服务：`Application/AdminBroadcastService.cs`

## 联动关系
- `Accounts`：通过 `AccountRepository` 校验管理员角色。
- `Notifications`：使用 `INotificationPublisher` 发送系统通知。
- `StrategyManagement`：通过 `StrategyRepository` 创建策略并切换策略实例状态。

## 超级测试台
### 权限与开关
- 仅超级管理员可访问（角色值由 `BusinessRules.SuperAdminRole` 控制，默认 `255`）。
- 功能总开关：`BusinessRules.EnableSuperAdminTestTools`（默认 `false`）。
- 当前端调用能力接口时，返回是否启用，便于前端隐藏入口。

### 接口
1. `POST /api/admin/user/test/capability`
- type: `admin.user.test.capability`
- 作用：返回超级测试台开关状态。
- 返回：`enabled: boolean`

2. `POST /api/admin/user/test/create-user`
- type: `admin.user.test.create-user`
- 入参：`email`、`password(>=6)`、`nickname?`
- 作用：创建普通角色测试账号（`role=0`），密码使用 BCrypt 哈希。
- 返回：`uid/email/nickname/role/status`

3. `POST /api/admin/user/test/create-random-strategy`
- type: `admin.user.test.create-random-strategy`
- 入参：
  - `targetUid`
  - `autoSwitchToTesting?`（默认 `true`）
  - `preferredExchange?`（可选，留空随机；支持：`binance|okx|bitget`）
  - `preferredTimeframeSec?`（可选，留空随机；支持：`60|300|900|3600|14400`）
  - `preferredSymbol?`（可选，留空随机；支持：`BTC/USDT|ETH/USDT|SOL/USDT|XRP/USDT`）
- 作用：
  - 使用全量 talib 指标池随机生成策略配置并创建到目标用户。
  - 支持按需固定交易所/时间周期/交易对，避免策略样本过于分散。
  - 创建后尝试切换策略实例到 `testing`。
  - 若切换失败，不回滚已创建策略，返回失败详情用于继续调试。
- 返回：
  - `strategy`: `defId/usId/versionId/versionNo/name/state`
  - `random`: `entryMethod/exitMethod/leftIndicator/rightIndicator/timeframe/exchange/symbol`
  - `testing`: `requested/success/message/state`

### 随机策略说明
- 指标来源：`Config/talib_indicators_config.json` + `talib_web_api_meta.json`（优先 `Config/`，回退 `Client/public/`）。
- 指标抽样范围：默认限制为常用指标池（如 `MACD/RSI/MA/EMA/SMA/BBANDS/ATR/ADX/CCI/STOCH/MFI/OBV/WILLR/ROC/MOM`），不再从全量 160 指标中无约束抽样。
- 参数来源：优先 meta 默认值，缺失时回退 common 默认值，`Integer/MAType` 按整数处理。
- 条件方法池：`GreaterThan/LessThan/CrossUp/CrossDown/CrossAny`，退出条件使用反向方法。
- 策略结构：`entry.long` 与 `exit.long` 各一组条件，`entry.short/exit.short` 关闭，`runtime.scheduleType=Always`。
