# ExchangeApiKeys 模块功能说明

## 功能概述
- 管理用户交易所 API Key（列表、绑定、解绑）。
- 统一交易所类型与 Passphrase 校验规则。

## 主要入口
- 控制器：`Controllers/UserExchangeApiKeysController.cs`
- 仓储：`Infrastructure/UserExchangeApiKeyRepository.cs`
- 领域模型：`Domain/UserExchangeApiKeyModels.cs`

## 联动关系
- `Accounts`：通过 `AuthTokenService` 进行用户鉴权。
- `StrategyEngine`/`TradingExecution`/`StrategyRuntime`/`StrategyManagement`：读取 API Key 用于策略运行或交易执行。
