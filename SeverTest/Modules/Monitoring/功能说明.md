# Monitoring 模块功能说明

## 功能概述
- 提供系统启动监控宿主抽象与运行状态日志采集能力。
- 主服务默认使用无 UI 空实现，避免对 Windows/WinForms 产生硬依赖。
- 汇总关键模块的启动与运行状态，便于运维与排障。

## 配置选项
- `MonitoringOptions.EnableDesktopHost`：是否尝试启用进程内桌面监控（仅 Windows 有效，默认 `false`）。
- `MonitoringOptions`：监控窗口日志与交易日志的缓存上限。

## 主要入口
- 监控宿主接口与实现：`Application/IStartupMonitorHost.cs`、`Application/NoopStartupMonitorHost.cs`、`Application/WinFormsStartupMonitorHost.cs`
- 监控窗体相关代码（保留，主服务默认不编译）：`Application/StartupMonitorHost.cs`、`Application/StartupMonitorForm.cs`
- 日志提供器：`Infrastructure/StartupMonitorLoggerProvider.cs`
- 领域模型：`Domain/StartupMonitorLogEntry.cs`

## 联动关系
- `MarketData`：展示历史行情缓存与下载状态。
- `MarketStreaming`：展示行情流与连接状态（通过 `IConnectionManager`）。
- `StrategyEngine`：展示策略引擎相关状态（由启动管理器收集）。
