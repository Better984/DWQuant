用户设置接口文档

1. 修改昵称与签名（WebSocket）

WS 消息类型：account.profile.update

请求：
{
  "type": "account.profile.update",
  "reqId": "req-1",
  "ts": 1700000000000,
  "payload": {
    "nickname": "新昵称",
    "signature": "新签名"
  },
  "err": null
}

响应：
{
  "type": "account.profile.updated",
  "reqId": "req-1",
  "ts": 1700000000000,
  "payload": {
    "success": true,
    "nickname": "新昵称",
    "signature": "新签名"
  },
  "err": null
}
